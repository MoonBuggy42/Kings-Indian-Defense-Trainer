# King's Indian Defense Trainer

This repository now contains only the static website for the King's Indian Defense trainer.

## Files
- `index.html` — the main web page
- `app.js` — application logic using Stockfish and chess.js
- `stockfish.js` — Stockfish engine worker
- `style.css` — UI styling
- `manifest.json` — PWA metadata
- `chess.js` — local chess library

## Notes
- This site can be hosted with GitHub Pages or any static web server.
- No Gradle, Java, or Android tooling is required.
